The User Portal is where paid users manage their AI Puffer account.

Use it to download the plugin, manage licenses, update billing details, view invoices, and change subscriptions.

Open the portal here: [users.freemius.com](https://users.freemius.com)

## Portal access

After checkout, Freemius sends an email with your portal access details.

If you do not have the password, use **Forgot Password** on the portal login page.

> Screenshot placeholder: User Portal login screen with the Forgot Password link.

## Websites

Use **Websites** to see where AI Puffer is installed and which site has an active license.

Open: [Manage Websites](https://users.freemius.com/websites)

You can:

- View connected websites.
- Deactivate a license from a website.
- Reuse the license on another website after deactivation.

> Screenshot placeholder: User Portal Websites screen showing installed websites.

### Move a license to another site

1. Open **Websites**.
2. Select the website that currently uses the license.
3. Select AI Puffer if the site has more than one product.
4. Open the license details.
5. Click **Deactivate**.

> Screenshot placeholder: Website details panel showing the license Deactivate button.

After deactivation, activate the same license on the new WordPress site.

## Downloads

Use **Downloads** to download the latest Pro ZIP file or an older version.

Open: [Downloads](https://users.freemius.com/downloads)

To download an older version:

1. Open **Downloads**.
2. Find AI Puffer.
3. Open the version menu next to the download button.
4. Select the version you need.

> Screenshot placeholder: Downloads screen showing AI Puffer and the version dropdown.

Beta versions, when available, appear in the same version menu with a **Beta** label.

## Updates

Plugin updates appear in the WordPress admin dashboard.

As long as your license is active, you can update the Pro plugin from **Dashboard > Updates** or **Plugins**.

> Screenshot placeholder: WordPress Plugins screen showing an available AI Puffer update.

## Licenses

Use **Licenses** to view license keys and check license status.

Open: [Manage Licenses](https://users.freemius.com/licenses)

## Upgrade a license

Use this when you want to move from a single-site license to a license that covers more sites.

1. Open **Renewals & Billing**.
2. Select the subscription.
3. Open the plan or license upgrade menu.
4. Choose the new license.
5. Complete checkout.

> Screenshot placeholder: Subscription details panel showing the upgrade menu.

Freemius prorates the payment based on the current subscription.

## Payments and invoices

Use **Payments** to view payment history, download invoices, or update payment details.

Open: [Payments](https://users.freemius.com/payments)

To download an invoice:

1. Open **Payments**.
2. Find the payment.
3. Click **Invoice**.

> Screenshot placeholder: Orders History screen showing the Invoice button.

## Add VAT or business details

1. Open **My Profile**.
2. Update your billing information.
3. Add your Tax or VAT ID.
4. Save the profile.

Open: [Profile Settings](https://users.freemius.com/profile)

Future invoices will include the updated billing details.

> Screenshot placeholder: Profile billing fields with Tax or VAT ID entered.

## Update payment method

1. Open **Renewals & Billing**.
2. Select the subscription.
3. Click **Update** next to the payment method.
4. Complete checkout with the new card or PayPal account.

> Screenshot placeholder: Subscription panel showing the Update payment method button.

## Cancel auto-renewal

1. Open **Renewals & Billing**.
2. Select the active subscription.
3. Click **Cancel Auto-Renew**.
4. Confirm the cancellation.

> Screenshot placeholder: Subscription details panel showing the Cancel Auto-Renew button.

Canceling auto-renewal stops future renewals. The license remains usable according to the terms shown in the portal.
