# AI Puffer docs

This repository contains the Mintlify documentation site for AI Puffer.

## Local preview

Install the Mintlify CLI:

```bash
npm i -g mint
```

Run the preview from this repository root:

```bash
mint dev
```

## Checks

Check links before publishing:

```bash
mint broken-links
```

## Deploy

Mintlify deploys from the `main` branch through the connected GitHub repository.
