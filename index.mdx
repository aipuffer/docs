AI Puffer adds AI features to WordPress through separate modules for chat, writing, automation, forms, media generation, knowledge sources, and usage controls.

Use these docs to configure each module and find the settings that affect it.

> Screenshot placeholder: AI Puffer dashboard home screen.

## Start here

1. Configure at least one provider in [AI Providers](/ai-providers).
2. Open the module you want to use.
3. Select the provider and model for that module.
4. Save the module settings.
5. Add the shortcode, block, or workflow to your WordPress site.
