AI Puffer is a WordPress plugin that adds AI tools to your website and admin area.

You connect your own AI provider keys, choose which model each feature should use, and use the tools through WordPress pages, shortcodes, blocks, products, and scheduled tasks.

You can use AI Puffer to add a chatbot, write posts and product content, build AI forms, generate images, run recurring automations, index knowledge sources, and track usage.

The main modules are Chat, Content Writer, Automations, AI Forms, Image / Video Generator, Knowledge Base / Training, and Usage & Billing. Each module has its own settings, but they share the same provider connections and usage records.

AI Puffer supports providers and services such as OpenAI, Google, Azure, OpenRouter, DeepSeek, Ollama, Claude, ElevenLabs, Replicate, Pinecone, and Qdrant. Availability depends on the module and the provider features you enable.

Usage & Billing records requests made through the plugin. It can be used for logs, limits, pricing rules, user balances, and WooCommerce credit packages.
