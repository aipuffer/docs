# Documentation project instructions

## About this project

- This is a Mintlify documentation site for AI Puffer.
- Pages are MDX files with YAML frontmatter.
- Configuration lives in `docs.json`.
- Run `mint dev` to preview locally.
- Run `mint broken-links` to check links.

## Terminology

- Use **AI Puffer** as the product name.
- Do not use the old product name in customer-facing docs.
- Use **credits** for the internal AI Puffer balance units.
- Use **quota** for periodic usage limits.

## Style

- Keep language concise and direct.
- Explain what a feature is and how to use it.
- Avoid promotional wording.
- Use numbered steps for setup flows.
- Use screenshot placeholders where screenshots are still needed.
- Bold UI labels, for example: Click **Settings**.
